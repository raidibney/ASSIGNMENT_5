    - 1️⃣ What is the difference between var, let, and const?
     Ans- 
    
   var- is the old way of declaring variables before ES6. The key features of the var is it is function scoped , it can be redeclared and reassigned.

   Let- let introduced in ES6 and it is the modern way of declearing a variable .it is blocked scoped , it cannot be redeclared in the same scope but can be reassigned.

   const- it is used for values that should not change.it is blocked scoped and it cannot be redeclared or reassigned.

    - 2️⃣ What is the spread operator (...)?
       Ans- 

   The spread operator in javascript is used to expand or spread the elements of and object or an array into another array or object.It is commonly used for copying arrays/objects or merging them.

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
         Ans-
    
   map() - is used to transform each element of an array. It applies a function to every element and returns a new array with the modified values.
             
   filter()- is used to select elements based on a condition. It checks each element and returns a new array containing only the elements that meet the condition.

   forEach()= is used to loop through each element of an array and perform an action. It does not return a new array; it simply executes the function for each element.

    - 4️⃣ What is an arrow function?
        Ans-
  
   An arrow function is a shorter way to write a function in JavaScript using the => syntax. It makes the code more concise and easier to read, especially for simple functions.

    - 5️⃣ What are template literals?
         Ans-

   Template literals are a way to write strings in JavaScript using backticks ( ) instead of quotes. They allow you to embed variables and expressions inside a string using ${ }.
